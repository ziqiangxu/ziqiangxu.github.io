``` shell
sudo ssh -L 443:127.0.0.1:8888 daryl@123.123.1.1
将本地443端口的请求转发到远程主机123.123.1.1的8888端口上
```
这在远程主机上使用`jupyter notebook`很有用，你就不需要把服务器的8888端口暴露在公网上
